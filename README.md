# Vite template for front end

Development is designed on Vite
Development is designed on Vite

### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ npm install
```

### Development

Open your favorite Terminal and run this command.

```sh
$ npm run dev
```

### Build

Open your favorite Terminal and run this command.

```sh
$ npm run build
```

### Preview

Open your favorite Terminal and run this command to preview build in browser.

```sh
$ npm run preview
```
